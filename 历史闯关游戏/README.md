# 变法风云 - 中国历代改革互动闯关游戏

## 📖 项目简介

这是一个基于人教版高中历史选择性必修1"国家制度与社会治理"第四课《中国历代变法和改革》设计的互动闯关游戏课件。

### 内容涵盖

- **第一关**：商鞅变法（战国时期）
- **第二关**：北魏孝文帝改革（南北朝时期）
- **第三关**：王安石变法（北宋时期）
- **第四关**：戊戌变法（晚清时期）
- **第五关**：张居正改革（明朝时期）
- **第六关**：改革开放（当代中国）

## 🎮 游戏特色

### 多样化题型

- **选择题**：考查历史知识的理解与记忆
- **判断题**：快速判断历史陈述的正误
- **填空题**：填写关键历史信息
- **排序题**：按时间顺序排列历史事件
- **连线题**：将历史人物与事件、措施正确配对

### 互动功能

- ⏱️ 限时答题系统
- 🔥 连击计分系统
- ⭐ 星级评价体系
- 📊 进度保存功能
- 🎬 历史视频资源
- 🖼️ 历史图片画廊

### 精美设计

- 采用古铜金配深红色的历史配色
- 中国传统书法风格的字体
- 精美的启动动画和过渡效果
- 响应式设计，适配各种设备

## 🚀 如何使用

1. 直接在浏览器中打开 `index.html` 文件
2. 点击"开始闯关"进入主菜单
3. 选择关卡开始答题挑战
4. 每关包含章节介绍、视频资源和答题环节
5. 完成答题后查看成绩和星级评价

## 📱 设备适配

- 💻 电脑端：最佳体验
- 📱 手机端：优化触摸操作
- 📟 平板端：横屏/竖屏均可

## ⚙️ 游戏规则

- 每题基础分：100分
- 时间奖励：剩余秒数 × 2
- 连击加成：第N次连击 + (N-1) × 10分
- 初始生命：3条
- 星级阈值：
  - ⭐：正确率 ≥ 60%
  - ⭐⭐：正确率 ≥ 80%
  - ⭐⭐⭐：正确率 ≥ 95%

## 📚 学习资源

每关都包含相关历史纪录片视频链接和历史图片，帮助学生更好地理解历史内容。

## 💾 数据保存

游戏进度自动保存在浏览器本地存储中，刷新页面后数据不会丢失。

## 🎯 教育意义

通过互动闯关的形式，激发学生学习历史的兴趣，加深对"中国历代变法和改革"这一主题的理解，认识到改革创新对社会发展的重要推动作用。
